## Fonctionnalités

Dans cette conférence, je montre comment ajouter des fonctionnalités facilement à WordPress grâce aux composants React

- SlotFills
- Blocs WordPress
- Page d'option
- DataFields / DataForms

## Installation 

- Ajoutez l'extension dans le dossier `wp-content/plugins/`de votre site local.
- Lancez la commande `npm install` la première fois uniquement.
- Lancez la commande `npm start`pour lancer le compilateur
